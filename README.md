Skeleton.js
===================
A Spinal chord for a Node.js Application


### Running the server

For development server:
```bash
node app.js 
```

For production:

```bash
NODE_ENV=production node app.js
```


### Prerequisites
```bash
npm install
bower install
grunt combine
```
